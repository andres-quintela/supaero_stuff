\subsection{Detailed Description}
Simple definition of graphs with vertices containing {\ttfamily char} values. \begin{DoxyAuthor}{Author}
Christophe Garion
\end{DoxyAuthor}
This is a simple definition of a graph containing {\ttfamily char} values implemented using adjacency lists represented by arrays. Warning\-: the vertices pointers should not be shared as deallocation of a graph deallocates the memory space reserved for the vertices!

The A\-P\-I of the graph is defined as follows\-:


\begin{DoxyItemize}
\item a function to create an empty graph
\item a function to add a vertex
\item a function to get a pointer to a vertex
\item a function to add an edge
\item a function to compute shortest paths from a given vertex to all other vertices in the graph
\item a function to print a graph
\item a function to deallocate a graph 
\end{DoxyItemize}
