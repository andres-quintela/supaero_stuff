Detailed Description

Simple definition of graphs with vertices containing char values.

Author
Christophe Garion

This is a simple definition of a graph containing char values implemented using adjacency lists represented by arrays. Warning: the vertices pointers should not be shared as deallocation of a graph deallocates the memory space reserved for the vertices!

The API of the graph is defined as follows:

  • a function to create an empty graph
  • a function to add a vertex
  • a function to get a pointer to a vertex
  • a function to add an edge
  • a function to compute shortest paths from a given vertex to all other vertices in the graph
  • a function to print a graph
  • a function to deallocate a graph
